NextGEN TinyMce Description add tinymce to nextgen gallery description.

== Description ==

NextGEN TinyMce Description simply integrate tinymce wysiwyg to the description of the gallery in NextGEN gallery manager.

== Frequently Asked Questions ==

== Installation ==

1. 	Copy the plugin in wp-content/plugins directory

2.	Activate the plugin

3. 	Go to your "manage gallery" on NextGEN, you will see description textarea modified. There is no configuration panel.

That's it ... Have fun
